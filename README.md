# Padua / IMPROVE bleeding risk score 計算補助ツール

入院中の内科系患者におけるVTEリスク評価と出血リスク評価を補助するための簡易計算ツールです。

## 使用目的

Padua Prediction ScoreおよびIMPROVE Bleeding Risk Scoreの点数計算とリスク分類を表示します。

## 注意事項

- 本ツールは医師向けの計算補助ツールです。
- 患者氏名、ID、生年月日などの個人情報は入力しないでください。
- 薬物的DVT予防の開始・中止・薬剤選択・用量は提示しません。
- 最終判断は担当医が臨床状況を踏まえて行ってください。
